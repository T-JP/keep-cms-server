# keep-cms-server

#### 介绍
一个简单的内容管理系统服务端


#### 软件架构
keep-cms的server服务端,使用node.js和mongodb搭建,web框架为koa.js.


#### 安装教程

1. npm i

#### 使用说明

1. npm run dev

#### 参与贡献

1. Fork 本仓库
2. 新建 Feat_xxx 分支
3. 提交代码
4. 新建 Pull Request


#### 目录说明

config 所有配置项
db 连接数据库基本操作
entity 表和增删改查操作
feature 功能点
koa web框架
log 打印信息
middleware koa中间件
plugin koa插件
route koa路由
util 项目工具
